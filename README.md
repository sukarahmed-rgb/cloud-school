# Cloud School (كلاود سكول)

![License](https://img.shields.io/badge/license-MIT-blue)

منصة تعليمية سحابية متكاملة مصممة خصيصًا لتلبية احتياجات الطلاب المكفوفين وضعاف البصر.

## المميزات
- **مساعد صوتي ذكي:** يقرأ المناهج والأسئلة بوضوح (Gemini TTS + Browser Speech)
- **محاكي كيبورد برايل (Perkins):** يتيح للطلاب الكتابة باستخدام نقاط برايل الست
- **واصف الرسوم البيانية:** رفع الصور ليقوم الذكاء الاصطناعي بوصفها صوتيًا
- **متعدد اللغات (i18n):** يدعم اللغتين العربية والإنجليزية
- **اختبارات تفاعلية وألعاب:** لتعزيز الاستيعاب
- **دعم PWA:** يعمل كتطبيق ويب قابل للتثبيت

## هيكلة المشروع

```
cloud_school/
├── src/                    # كود المصدر المهيكل إلى وحدات
│   ├── braille.js          # لغة برايل
│   ├── errors.js           # معالج الأخطاء المركزي
│   ├── firebase.js         # Firebase والتخزين السحابي
│   ├── gemini.js           # استدعاءات الذكاء الاصطناعي
│   ├── i18n.js             # الترجمة والدولية
│   └── ui.js               # أدوات واجهة المستخدم
├── tests/
│   ├── unit/               # اختبارات الوحدة (Jest)
│   └── e2e/                # اختبارات الواجهة (Playwright)
├── i18n/                   # ملفات الترجمة
├── proxy/                  # الخادم الوسيط (لحماية مفتاح API)
├── index.html              # الصفحة الرئيسية
├── cloud_school_app.js     # ملف التطبيق الرئيسي
├── cloud_school.js         # Service Worker
├── cloud_school_styles.css # الأنماط
├── firestore.rules         # قواعد أمان Firebase
└── .env.example            # مثال لمتغيرات البيئة
```

## البدء السريع

### المتطلبات الأساسية
- متصفح حديث (Chrome موصى به)
- Node.js 18+ (للسيرفر الوسيط والاختبارات)

### التشغيل للتطوير

```bash
# تثبيت الاعتماديات
npm install

# تشغيل الخادم المحلي
npm run dev
```

### تشغيل الخادم الوسيط (لحماية مفتاح API)

```bash
# 1. انسخ .env.example إلى .env وأضف مفتاح Gemini
# 2. شغّل الخادم الوسيط
npm run proxy
```

### الاختبارات

```bash
# اختبارات الوحدة
npm test

# اختبارات E2E
npm run test:ui
```

## النشر

المنصة جاهزة للنشر على GitHub Pages. عند دفع التغييرات إلى `main`، يقوم CI بتشغيل الاختبارات ونشر النسخة.

## الأمان

- لا تضع مفتاح API في كود المصدر أبداً
- استخدم الخادم الوسيط (`proxy/server.js`) في الإنتاج
- راجع `firestore.rules` لإعداد صلاحيات Firebase
- الـ CSP (Content Security Policy) مضمن في `index.html`

---

*بُني هذا النظام لخدمة ذوي الهمم وتوفير بيئة تعليمية متطورة.*
