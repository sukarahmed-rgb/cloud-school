# Cloud School (كلاود سكول)

![Build Status](https://img.shields.io/github/actions/workflow/status/username/cloud-school/ci.yml?branch=main)
![License](https://img.shields.io/badge/license-MIT-blue)

منصة تعليمية سحابية متكاملة مصممة خصيصًا لتلبية احتياجات الطلاب المكفوفين وضعاف البصر. تعتمد المنصة على تقنيات الذكاء الاصطناعي (Gemini) لتحويل النصوص إلى صوت طبيعي، وتدعم إدخال وقراءة لغة برايل (محاكي Perkins)، وتوفر واجهة عالية التباين متوافقة مع معايير الوصول (WCAG).

## المميزات
- **مساعد صوتي ذكي:** يقرأ المناهج والأسئلة بوضوح.
- **محاكي كيبورد برايل (Perkins):** يتيح للطلاب الكتابة باستخدام نقاط برايل الست.
- **واصف الرسوم البيانية:** رفع الصور ليقوم الذكاء الاصطناعي بوصفها صوتيًا.
- **متعدد اللغات (i18n):** يدعم اللغتين العربية والإنجليزية.
- **اختبارات تفاعلية وألعاب:** لتعزيز الاستيعاب.
- **دعم PWA:** يعمل كتطبيق ويب قابل للتثبيت ومتاح دون إنترنت بفضل Service Workers و IndexedDB.

## البدء السريع (Quick Start)

### المتطلبات الأساسية
- متصفح حديث (Google Chrome موصى به).
- اتصال بالإنترنت (لجلب ملفات Firebase و Gemini).

### التشغيل محليًا (Local Development)
نظرًا لأن المشروع يعتمد على وحدات JavaScript (`type="module"`), يجب تشغيله عبر خادم ويب محلي:

1. **استخدام Live Server في VS Code:**
   انقر بزر الماوس الأيمن على `cloud_school.html` واختر "Open with Live Server".
   
2. **استخدام Node.js:**
   ```bash
   npm install -g live-server
   live-server .
   ```

## الاختبار (Testing)

تم إعداد المشروع لدعم اختبارات الوحدة (Unit) واختبارات الواجهة (E2E).

### تشغيل اختبارات الوحدة (Jest)
```bash
npm install
npm test
```

### تشغيل اختبارات E2E (Playwright)
يقوم Playwright بفتح متصفح Chrome للتحقق من تفاعلات الواجهة:
```bash
npm run test:ui
```

## النشر (Deployment)

المنصة معدة وجاهزة للنشر التلقائي عبر **GitHub Pages**.
عند دفع التغييرات إلى فرع `main`، سيقوم GitHub Actions بتشغيل الاختبارات تلقائيًا ونشر النسخة النهائية إذا اجتازت الاختبارات.

1. ادفع الكود إلى مستودعك على GitHub.
2. اذهب إلى **Settings > Pages**.
3. تأكد من أن الـ Source مضبوط على `GitHub Actions`.
